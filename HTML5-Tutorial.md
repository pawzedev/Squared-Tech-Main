## 01. What is HTML5?
`HTML5` is a **programming language** useful to build websites. The *extension* to *HTML5* is `.html`. We will explain more about extensions **soon**.
## 02. HTML5: Main Structure 🏗️
All `HTML5` scripts always have the same structure:
```
<!DOCTYPE html>
<html lang="">
<head>
<title>Website title here!</title>
</head>
<body>

</body>
</html>
```
This structure shows **all** the *main structure* tags: `head`, `title` and `body`.
### What are those tags?
The `<!DOCTYPE html>` line/tag **explains** to the program that runs *HTML5* that the script is **actually** programmed in **HTML**. The `<html>` tag explains where the **HTML** code starts and ends. Inside `<html>`, `lang=""` explains to the browser what language the code was programmed. Some examples are: `lang="en"` for **english**, `lang="pt-br"` for **brazilian portuguese**, etc. The `<head>` tag contains what happens "behind the scenes" of the website. The `body` tag contains the layout of the website, usually the **tag** with the most **tags** inside.
